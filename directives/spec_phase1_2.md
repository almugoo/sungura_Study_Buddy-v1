# Sungura AI Study Buddy - Project Specification

## Project Vibe & Personality
**Sungura** ("rabbit" in Swahili) - a fast, intelligent, and encouraging AI study companion for East African university students.
*   **Philosophy**: Speed, Simplicity, Warmth, Cultural Relevance, Empowerment.
*   **Aesthetic**: Clean, modern, sunrise colors, friendly illustrations, dark mode support.

## Technical Stack
*   **Frontend**: React Native (iOS + Android).
*   **Backend**: Node.js with Express.js.
*   **Database**: PostgreSQL (structured), MongoDB (logs/unstructured).
*   **Auth**: Firebase Auth.
*   **AI**: OpenAI GPT-4 or Anthropic Claude.
*   **Infrastructure**: AWS/GCP/DigitalOcean.

## PHASE 1: MVP Features
1.  **User Onboarding**: University selection, course addition, learning style quiz.
2.  **Personal AI Sherpa**: Chat interface (English/Swahili), Adaptive responses (Visual, Auditory, Kinesthetic, etc.), Subject support (Math, Science, etc.), Multi-modal (Text, Image/OCR, Voice).
3.  **Course Management Dashboard**: Mastery tracker, course cards.
4.  **Progress Tracking**: Study streaks, time tracking.
5.  **Infrastructure**: Offline mode, Data efficiency (<50MB app), Security.

## PHASE 2: Enhanced Learning & Collaboration
1.  **AI-Facilitated Study Groups**: Group creation, AI moderation/facilitation.
2.  **Advanced Diagnostic & Adaptation**: Knowledge graph, adaptive difficulty.
3.  **Rich Content Generation**: Dynamic diagrams, video recommendations, problem banks.
4.  **Exam Preparation Mode**: Countdown, study plans, past paper analysis, mock exams.

## Success Criteria (MVP)
*   1,000 active users.
*   70% user satisfaction.
*   85% AI accuracy.
*   Stable app.
